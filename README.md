# MOCK34 Chat Application

A modern, responsive chat application with a beautiful UI and fully functional chat interface.

## Features

✅ **Fully Working Chat Box** - Send and receive messages seamlessly
✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile
✅ **Beautiful UI** - Modern gradient design with smooth animations
✅ **Send Button** - Fully functional with click and Enter key support
✅ **Auto-scroll** - Messages automatically scroll to show the latest conversation
✅ **Bot Responses** - Simulated AI responses for testing

## Files

- `index.html` - Main chat interface structure
- `styles.css` - Complete styling and animations
- `script.js` - Chat functionality and event handlers

## How to Use

1. Open `index.html` in your browser
2. Type a message in the chat input box
3. Click the **Send** button or press **Enter** to send
4. Receive instant bot responses

## Technical Details

- Pure vanilla JavaScript (no dependencies)
- CSS Grid and Flexbox layout
- Smooth animations with CSS keyframes
- Mobile-first responsive design

## Deployment

Host the files on any static web server (GitHub Pages, Netlify, Vercel, etc.)
